<div className="bulma-tags">

</div>

**Type**: TSInterface

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|currentTarget|[InfiniteGrid](InfiniteGrid)|<p>An InfiniteGrid instance that triggered this event. </p>|
|element|HTMLElement|<p>The item's element.</p>|
|target|HTMLElement|<p>The content element with error.</p>|
|item|InfiniteGridItem|<p>The item with error content.</p>|
|update|() =&gt; void|<p>If you have fixed the error and want to recheck it, call update(). If you remove an element, call the syncElements() method.</p>|
