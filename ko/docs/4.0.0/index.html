Installation

Using the package managers (recommended)#

You can easily install InfiniteGrid with package managers like npm or yarn

npm#

  • JavaScript
  • React
  • Vue@2
  • Vue@3
  • Angular
  • Svelte
npm install @egjs/infinitegrid
npm install @egjs/react-infinitegrid
npm install @egjs/vue-infinitegrid
npm install @egjs/vue3-infinitegrid
npm install @egjs/ngx-infinitegrid
npm install @egjs/svelte-infinitegrid

yarn#

  • JavaScript
  • React
  • Vue@2
  • Vue@3
  • Angular
  • Svelte
yarn add @egjs/infinitegrid
yarn add @egjs/react-infinitegrid
yarn add @egjs/vue-infinitegrid
yarn add @egjs/vue3-infinitegrid
yarn add @egjs/ngx-infinitegrid
yarn add @egjs/svelte-infinitegrid
